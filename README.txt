EventFinder


Authors: Anuj Patel and Vijay Jayaram

Originally a project for 15-437 (Web Applications Development) at Carnegie Mellon. 
EventFinder is an app that allows users to create events at particular locations and find other events in their area.

Currently deployed on EC2 at this URL: ec2-54-191-107-23.us-west-2.compute.amazonaws.com
